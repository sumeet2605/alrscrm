# ALRSCRM Permission Matrix

## Roles
- Super Admin
- Owner
- Branch Manager
- Sales Executive
- Photographer
- Editor
- Customer Success
- Client

## Key Rules
Owner: Full access across branches.
Branch Manager: Full access within assigned branch.
Sales Executive: CRM, Opportunities, Follow-ups.
Photographer: Assigned sessions only.
Editor: Assigned editing jobs only.
Customer Success: Galleries, Selections, Deliveries.
Client: Portal access only.
